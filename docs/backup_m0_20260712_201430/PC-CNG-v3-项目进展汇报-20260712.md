# PC-CNG v3 项目进展汇报

日期：2026-07-12

## 一、项目定位

PC-CNG (Pairwise Counterfactual Chemical Negative Generation) 是一个**化学反应推理边界负样本生成与候选重排序**框架。核心贡献是通过边界反事实负样本 + pairwise reward 学习，在 same-context candidate reranking 任务上实现强性能。论文目标为顶刊发表。

## 二、已完成的核心工作

### 2.1 主模型（Type-1 same-context reranking）

- **基线模型**：Morgan fingerprint MLP + pairwise loss，overall Top-1 97.20%
- **v2/unreacted 升级**：加入 unreacted-substrate 负样本，test Top-1 从 85.07% → **87.16 ± 1.58%**（10 seeds）
- 代码：`pc_cng/train_pairwise_reward_mlp.py`

### 2.2 统计严谨性升级（顶刊级）

- **10-seed ensemble paired significance test**：两层统计检验
  - Group-level ensemble test：10 seed 取平均 → paired bootstrap CI + sign-flip permutation test + sign test
  - Seed-level bootstrap CI：seed index 重采样 → 跨 seed 稳健性
- 代码：`pc_cng/multiseed_paired_significance.py`
- 单测：`tests/test_multiseed_paired_significance.py`（4 项全部通过）

## 九、泛化 Gap 深度分析

**核心发现**：模型在训练集上已接近完美（train Top-1 99.62%），但泛化到 unseen groups 时下降明显（test Top-1 ~87%）。

**可能原因**：
1. **过拟合到训练 reaction 的特定模式**：Morgan fingerprint 对训练数据的子结构模式过拟合
2. **训练数据多样性不足**：1,060 个 training groups 可能不足以覆盖 test 分布
3. **Early stopping 指标不匹配**：训练用 val ROC-AUC（binary classification）选 best checkpoint，但目标是优化 Top-1（ranking）
4. **特征表示的局限性**：Morgan fingerprint 可能不足以捕捉某些化学反应的全局结构特征

**待验证的优化方向**：
| 方向 | 状态 | 预期 |
|---|---|---|
| 更大模型容量 (hidden=4096) | 进行中，7/10 seeds | 预期无效（初步趋势偏负） |
| 更强正则化 (dropout=0.4) | 进行中，3/10 seeds | 待验证 |
| Cosine LR scheduler + warmup | 脚本就绪，等 GPU 空闲 | 可能有效 |
| Early stopping 改为 val Top-1 | 待验证 | 可能有效 |
| 更大 n_bits (8192) | 待验证 | 不确定 |
| 更多样的合成数据 | 待验证 | 可能有效 |
| 数据增强 (SMILES augmentation) | 待验证 | 可能有效 |

## 十、Cosine LR Scheduler 实验

**实验目标**：验证 cosine learning rate decay + linear warmup 能否改善优化，提升 test Top-1。

**动机分析**：
- 当前使用固定 lr=1e-3，训练 80 epochs，无 scheduler
- 固定学习率可能在训练后期过冲，导致泛化不好
- Cosine annealing + warmup 是深度学习的标准最佳实践
- 预期：更平滑的收敛，可能提升泛化性能

**配置**：
- 架构：Morgan fingerprint MLP + pairwise loss
- hidden_dim: 2048, n_bits: 4096, dropout: 0.20（保持基线）
- lr: 1e-3 → cosine decay to 1e-5
- warmup: 5 epochs (linear warmup)
- batch_size: 4096, epochs: 80
- 数据：RegioSQM20 + HITEA (real) + 6 synthetic reviewed CSV
- 10 seeds: 20260710-20260719
- GPU: GPU 4（等 hidden4096 完成后启动）

**状态**：脚本已就绪，等待 GPU 4 空闲（hidden4096 完成后立即启动）
