# PC-CNG v3 下一阶段项目完善计划

## 科学问题

核心科学问题：

```text
在正反应数据稀缺、真实失败实验不足的情况下，
能否通过物理化学约束的反事实负样本生成，
学习反应可行性边界，
并提升下游 reaction outcome / product candidate ranking？
```

当前证据已经支持初步结论：

```text
PC-CNG hard negatives improve downstream candidate product reranking.
```

下一阶段目标是把证据链从 reranker 提升到 reaction LM / product prediction policy。

```text
- chemformer_forward

```

说明：

`length_baseline` 只验证 IO，不作化学结论。

`chemformer_forward` 是 top-k beam exact-match scorer：如果候选产物出现在 Chemformer top-10 beams 中，则使用 beam log-likelihood；否则给 unmatched penalty。该 scorer 已跑通，但 top-10 命中率很低：

```text
matched_beam: 43 / 32,287 candidates
not_in_top_beams: 32,244 / 32,287 candidates
```

因此它更适合作为“生成式 top-k 是否覆盖候选”的诊断，不适合作为最终 reranking 主证据。

`chemformer_log_likelihood` 是 teacher-forced conditional scorer：直接计算每个 candidate product 在给定 reactants/context 下的条件 log-likelihood。该 scorer 覆盖所有候选，更适合做 policy-level reranking 证据。
