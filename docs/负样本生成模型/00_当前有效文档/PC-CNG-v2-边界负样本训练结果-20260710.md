## v2 生成器变化

旧 MVP 主要生成：

- `product:=reactants`
- `retro_no_disconnection`
- `append:O`
- Br/Cl、N/O 字符串替换

v2 生成器改为：

- 使用 atom-mapped reaction center；
- 抽取 formed/broken/changed bonds；
- 对新形成键做局部迁移；
- 生成同一反应上下文下的 alternative product。

典型例子：

```text
N10-methylated observed product
-> N9-methylated alternative product
```

这更接近 Science Advances 论文中强调的 type-1 negative：意外但化学合理的竞争产物。

## 解释

v2 负样本还没有强到让单模型直接超过 real-only，但它提供了和 Morgan/descriptor/USPTO 模型不同的边界信号，因此在 ensemble 中带来了增益。

这说明：

1. PC-CNG v2 方向比旧 MVP 更正确；
2. 当前生成器仍然是规则型 reaction-center perturbation，不是最终 SOTA；
3. 下一步需要训练一个真正的 reaction-center edit decoder，而不是继续手写迁移规则；
4. 负样本应更多用于 reward / ranking / ensemble，而不是简单 BCE hard negative。
